SuddenSpark is a social platform that lets users share their spontaneous thoughts and ideas - those "shower thoughts" moments that strike when you least expect them. Built with modern web technologies, it offers a fun and engaging way to share and interact with bite-sized thoughts.

## Key Features

- 📝 Share thoughts in 160 characters or less
- 🎨 Colorful, eye-catching thought cards
- 👍❤️🤯 React to thoughts with various emojis
- 🌓 Dark mode support
- 🔍 Filter thoughts by popularity or recency
- 👤 User profiles to track thoughts by author

## Tech Stack

The application is built using:

- Next.js 13 with App Router for the frontend
- Firebase for backend and authentication
- Tailwind CSS for styling
- Framer Motion for smooth animations
- Google Gemini API for content moderation

The combination of these technologies allows for a responsive, real-time experience with seamless authentication and data persistence.
